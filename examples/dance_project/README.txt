# Open the folder for ltchin_ble_record_i2c_one_sensor or ltchin_ble_record_i2c_two_sensor for either the one sensor or two/multiple sensor versions, respectively.

* One sensor has the web interface for one sensor and does not include the MUX/selector code for more than one sensor

* Two sensor code can accommodate up to 8 sensors (MUX SCL/SDA ports 0-7)
